# QCalc - 생산성과 접근성을 위한 다목적 계산기

QCalc은 단순한 계산기를 넘어, 현대적인 생산성 도구로 진화시키려 합니다. Vue3(Quasar v2), TypeScript, Electron 등 최신 웹 기술을 활용하여 제작된 이 다목적 계산기는, 실제 업무, 작업, 학습 등의 상황에서 유용하게 활용할 수 있는 앱으로 발전시키고 있습니다. 또한 장애를 가진 사용자도 직관적으로 활용할 수 있도록 접근성에 집중하고 있으며, 간단하고 단순한 UI로 데스크톱과 모바일 환경 모두에서 유용한 사용자 경험을 제공할 수 있도록 고민하고 있습니다.

## 앱의 주요 특징

- **4가지 전문 계산기**: 표준 계산기, 단위 변환기, 통화 변환기, 진법 계산기로 다양한 계산 요구사항을 충족
- **고정밀도 계산 엔진**: 최대 64자리까지 정확한 계산을 지원하며, 삼각함수, 팩토리얼, N제곱 등 고급 수학 연산 제공
- **진법 변환 및 비트 연산**: 2/8/10/16진수 변환과 AND, OR, XOR, NOT 등 프로그래머를 위한 전문 비트 연산 지원
- **스마트한 사용자 경험**: 단위/통화 즐겨찾기, 다양한 색상 테마, 계산 기록 메모 기능으로 개인화된 사용 환경 제공
- **크로스 플랫폼 지원**: Windows, Linux 데스크톱과 Android 모바일에서 일관된 경험을 제공하며 자동 업데이트 지원
- **접근성 중심 설계**: 키보드 단축키, 햅틱 피드백, 적응형 레이아웃 등으로 모든 사용자가 쉽게 접근할 수 있도록 지속적 개선

## 주요 기능 사용 가이드

### 4가지 계산기 활용법

#### 표준 계산기

- **접근 방법**: Ctrl+1 또는 상단 탭 선택
- **기본 연산**: 0-9, +, -, \*, / 키로 입력
- **고급 기능**: 삼각함수(q, w, e), 제곱(u), 제곱근(i), 상수(z: π, x: φ, c: e)
- **메모리 기능**: Ctrl+Enter(MS), Ctrl+Backspace(MR), Ctrl+Delete(MC)
- **백분율 계산**: '숫자, /, 숫자, %(k)'는 백분율 계산
- **백분율 적용**: '숫자, \*, 숫자, %(k)'는 백분율 적용

#### 단위 변환기

- **접근 방법**: Ctrl+2 또는 상단 탭 선택
- **변환 범주**: 길이, 면적, 부피, 무게, 각도 등 15개 이상 카테고리
- **즐겨찾기**: 자주 사용하는 단위를 즐겨찾기로 설정하여 빠른 접근 가능
- **단위 교환**: '\' 키로 원본/변환 단위 교환 가능
- **빠른 변환**: ×10/×100/×1000(a/s/d), ÷10/÷100/÷1000(z/x/c)
- **단위 기호**: Alt+\ 키로 단위 기호 표시/숨김 전환

#### 통화 변환기

- **접근 방법**: Ctrl+3 또는 상단 탭 선택
- **최신 환율**: 실시간 환율 정보 반영
- **즐겨찾기**: 자주 사용하는 통화를 즐겨찾기로 설정하여 빠른 접근 가능
- **빠른 계산**: +5/+10/+100(f/g/h), -5/-10/-100(q/w/e)
- **통화 교환**: '\' 키로 원본/변환 통화 교환 가능
- **통화 기호**: Alt+\ 키로 통화 기호 표시/숨김 전환

#### 프로그래머 계산기

- **접근 방법**: Ctrl+4 또는 상단 탭 선택
- **지원 진법**: 2진수, 8진수, 10진수, 16진수 변환
- **16진수 입력**: A-F는 z, x, c, a, s, d 키로 입력
- **진법 교환**: '\' 키로 원본/변환 진법 교환 가능
- **진법 기호**: Alt+\ 키로 단위 기호 표시/숨김 전환
- **기호 위치**: Alt+Ctrl+\ 키로 단위 기호 위치(앞/뒤) 전환

### 생산성 향상 기능 활용법

#### 계산 기록 관리

- **접근 방법**: F4 키 또는 사이드 메뉴
- **스크롤**: ↑/↓ 키로 50px, Page Up/Down으로 400px 이동
- **검색/삭제**: Ctrl+F로 검색, Ctrl+D로 기록 삭제
- **메모 추가**: 개별 기록에 메모 추가 가능
- **왼쪽 스와이프(모바일)**: 메모 추가/편집
- **오른쪽 스와이프(모바일)**: 기록 삭제

#### 숫자 표시 설정

- **구분자 표시/숨김**: , 키로 전환
- **구분 단위 설정**: Alt+, 키로 3자리/4자리 변경
- **소수점 자릿수**: [, ] 키로 조정(무제한~16자리)

#### 단축키 활용

- **시프트 모드**: ' 키로 활성화하여 고급 기능 접근
- **탭 이동**: Ctrl+Tab(→), Ctrl+Shift+Tab(←)으로 탭 간 이동
- **화면 전환**: F1(도움말), F2(정보), F3(설정), F4(기록), F5(팁)

#### 복사 및 붙여넣기

- **메인 패널 복사**: Ctrl+C, Ctrl+Insert
- **보조 패널 복사**: Shift+Ctrl+C, Alt+Ctrl+Insert
- **메인 패널 붙여넣기**: Ctrl+V, Shift+Insert
- **보조 패널 붙여넣기**: Shift+Ctrl+V, Alt+Shift+Insert
- **메뉴 열기**: 패널을 클릭 후 메뉴에서 복사/붙여넣기

### **전문가용 도구 활용 팁**

#### 수학 기능

- **N제곱/N제곱근**: r/t 키로 계산
- **삼각함수**: 시프트 모드에서 q/w/e 키
- **정수/소수 추출**: v/b 키로 계산
- **팩토리얼**: h 키로 계산

#### 메모리 활용하기

- **메모리 저장&호출**: 메모리에 저장(MS)하고 계산 후 호출(MR)
- **메모리 계산**: M+, M-, M×, M÷ 기능으로 값 누적 처리
- **메모리 지움**: MC로 메모리 지움
- **메모리 상태**: 매인 패널의 메모리 아이콘 클릭

#### 비트 연산 활용

- **비트 연산**: AND(j), OR(k), XOR(l), NOT(h), 시프트 연산(r/t/u/i)
- **논리 확장**: NAND/NOR/XNOR 연산은 q/w/e 키로 사용
- **시프트 연산**: 비트 위치 이동에 1비트/4비트
- **비트 크기 설정**: 설정 크기에 띠라 비트 연산

### **사용자 경험 최적화**

#### 화면 레이아웃

- **테마 시스템**: 다크/라이트 모드를 넘어 다양한 색상 테마 선택 가능 (F3 설정에서 변경)
- **항상 창의 위에 위치**: Alt+t 키로 항상 창의 위에 위치 토글
- **창 크기 조절**: 화면 크기에 따라 사이드 패널 자동 조정
- **패널 초기화**: Alt+i 키로 시작 시 패널 초기화 토글
- **다크 모드 (비)활성화**: Alt+d 키로 다크 모드 전환

#### 모바일 지원

- **스와이프**: 좌우 스와이프로 계산기 모드 전환
- **햅틱 모드 (비)활성화**: Alt+p 키로 햅틱 모드 전환
- **태블릿 가로 모드**: 태블릿의 가로모드에서 사이드 패널 자동 조정

## 설치 방법

- 릴리즈 페이지에서 해당 플랫폼에 맞는 파일을 받아서 설치하면 됩니다.
- 윈도우 설치 파일과 리눅스 AppImage 파일은 자동 업데이트가 지원됩니다.
- 리눅스는 AppImage 파일을 단독으로 쓰거나 `sudo snap install --beta qcalc` 명령어로 설치하세요.
- 안드로이드는 해당 apk를 받아 설치하세요. 곧 플레이스토어에 올라갑니다.
- 맥OS, IOS는 사용하지 않고 테스트할 방법이 없어서 패키지가 없습니다.
