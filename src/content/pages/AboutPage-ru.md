## [0.12.0] 2026-03-22

### Добавлено

- **Калькулятор формул (5-й калькулятор)**: Вводите и вычисляйте математические выражения напрямую — поддерживает арифметику, скобки, функции (`sin`, `cos`, `sqrt`, `log`, `ln`, `abs`, `round`, `nthRoot` и др.) и константы (`pi`, `e`, `phi`) через синтаксис [mathjs](https://mathjs.org/).
  - Нажмите Пробел, чтобы открыть встроенный редактор формул для прямого редактирования выражений.
  - Используйте `@` для ссылки на текущее значение, `$` для значения из памяти.
  - Полная поддержка памяти (MC, MR, MS, M+, M−, M×, M÷) доступна через кнопки функций shift.
  - Результаты вычислений сохраняются в истории расчётов с отображением полного выражения.
  - Встроенное меню справки содержит список всех доступных функций, констант и заполнителей.
- **5 новых языков (всего 8)**: Китайский (упрощённый), Хинди, Немецкий, Испанский и Французский присоединяются к существующим Корейскому, Английскому и Японскому. Все экраны переведены — меню, настройки, названия единиц измерения, названия валют, страницы справки, страницы «О приложении», советы и сообщения об ошибках.
- **Упаковка Flatpak**: Установите QCalc через Flatpak для более широкой поддержки рабочих столов Linux.
- **Переход на бесплатный публичный API курсов валют**: Переход с FreeCurrencyAPI (требуется API-ключ) на бесплатные публичные API (Frankfurter + fawazahmed0). Поддерживает 340 валют (фиатные, драгоценные металлы, криптовалюты) без необходимости API-ключа. Снимки курсов во время сборки обеспечивают конвертацию валют даже при первом запуске без сети.

### Изменено

- **Более плавное переключение языка**: Если перевод отсутствует, приложение теперь автоматически переключается на английский вместо отображения необработанных путей ключей.
- **Расширенное покрытие валют**: С 170 до 340 валют — добавлены основные криптовалюты (ETH, SOL, XRP и др.), драгоценные металлы (палладий, платина) и исторические валюты. Полная поддержка i18n для всех 8 языков.

Для получения информации о предыдущих версиях, пожалуйста, перейдите [сюда](https://github.com/from104/qcalc/blob/main/CHANGELOG.md).
