# QCalc - Многофункциональный калькулятор для продуктивности и доступности

Высокоточные вычисления, конвертация единиц/валют/систем счисления и вычисление формул в одном приложении. Многофункциональный калькулятор с единообразным интерфейсом на настольных и мобильных устройствах.

## Основные возможности приложения

- **5 профессиональных калькуляторов**: Стандартный калькулятор, конвертер единиц, конвертер валют, калькулятор для программистов и калькулятор формул удовлетворят все ваши потребности в вычислениях
- **Высокоточный вычислительный движок**: Поддерживает точные вычисления до 64 знаков с продвинутыми математическими функциями — тригонометрия, факториал, степени
- **Системы счисления и битовые операции**: Конвертация между двоичной/восьмеричной/десятичной/шестнадцатеричной системами и поддержка профессиональных битовых операций (AND, OR, XOR, NOT) для программистов
- **Умный пользовательский опыт**: Персонализированная среда с избранными единицами/валютами, разнообразными цветовыми темами, заметками в истории вычислений и экспортом/импортом
- **Кроссплатформенная поддержка**: Единообразный интерфейс на Windows, Linux и Android с поддержкой автоматического обновления
- **Дизайн с фокусом на доступность**: Постоянное улучшение для удобства всех пользователей — сочетания клавиш, тактильная обратная связь и адаптивные макеты
- **Управление настройками**: Экспорт или импорт всех настроек в файл для сохранения одинаковых параметров в разных средах

## Руководство по основным функциям

### Как использовать 5 калькуляторов

#### Стандартный калькулятор

- **Как открыть**: Ctrl+1 или выберите верхнюю вкладку
- **Базовые операции**: Ввод чисел 0-9, клавиши +, -, \*, /
- **Расширенные функции**: Тригонометрические функции (q, w, e), квадрат (u), квадратный корень (i), константы (z: π, x: φ, c: e)
- **Функции памяти**: Ctrl+Enter (MS), Ctrl+Backspace (MR), Ctrl+Delete (MC)
- **Вычисление процента**: 'число, /, число, %(k)' вычисляет процент
- **Применить процент**: 'число, \*, число, %(k)' применяет процент

#### Конвертер единиц

- **Как открыть**: Ctrl+2 или выберите верхнюю вкладку
- **Категории конвертации**: Более 15 категорий, включая длину, площадь, объём, вес, угол и др.
- **Избранное**: Добавляйте часто используемые единицы в избранное для быстрого доступа
- **Поменять единицы**: Используйте клавишу '\' для обмена исходных/целевых единиц
- **Быстрая конвертация**: ×10/×100/×1000 (a/s/d), ÷10/÷100/÷1000 (z/x/c)
- **Символ единицы**: Переключение отображения символа единицы клавишей Alt+\

#### Конвертер валют

- **Как открыть**: Ctrl+3 или выберите верхнюю вкладку
- **340 валют**: Фиатные валюты, криптовалюты и драгоценные металлы — API-ключ не требуется
- **Актуальные курсы**: Курсы в реальном времени (при отсутствии сети используется встроенный снимок)
- **Избранное**: Добавляйте часто используемые валюты в избранное для быстрого доступа
- **Быстрый расчёт**: +5/+10/+100 (f/g/h), -5/-10/-100 (q/w/e)
- **Поменять валюты**: Используйте клавишу '\' для обмена исходной/целевой валюты
- **Символ валюты**: Переключение отображения символа валюты клавишей Alt+\

#### Калькулятор для программистов

- **Как открыть**: Ctrl+4 или выберите верхнюю вкладку
- **Поддерживаемые системы счисления**: Конвертация между двоичной, восьмеричной, десятичной и шестнадцатеричной
- **Ввод шестнадцатеричных цифр**: Ввод A-F клавишами z, x, c, a, s, d
- **Поменять системы**: Используйте клавишу '\' для обмена исходной/целевой системы
- **Символ системы**: Переключение отображения символа системы клавишей Alt+\
- **Позиция символа**: Переключение позиции символа (перед/после) клавишей Alt+Ctrl+\

#### Калькулятор формул

- **Как открыть**: Ctrl+5 или выберите верхнюю вкладку
- **Ввод выражений**: Вводите математические выражения напрямую, используя синтаксис mathjs
- **Поддерживаемые операции**: Все математические функции mathjs, включая арифметику, степени, тригонометрию, логарифмы
- **Ссылка на текущее значение**: Используйте символ @ для включения текущего вычисленного значения в выражения
- **Функции памяти**: Ctrl+Enter (MS), Ctrl+Backspace (MR), Ctrl+Delete (MC)
- **Режим редактирования**: Нажмите Пробел или Enter (когда пусто) для входа, Esc для очистки и выхода
- **Навигация по истории**: Используйте клавиши ↑/↓ в режиме редактирования для просмотра предыдущих выражений, автосброс при редактировании
- **Загрузить выражение**: Щёлкните правой кнопкой мыши по записи формулы → «Загрузить в поле формулы» для повторного использования выражений

### Как использовать функции продуктивности

#### Управление историей вычислений

- **Как открыть**: Клавиша F4 или боковое меню
- **Прокрутка**: Перемещение на 50px клавишами ↑/↓, на 400px клавишами Page Up/Down
- **Поиск/Удаление**: Поиск с Ctrl+F, удаление истории с Ctrl+D
- **Экспорт/Импорт**: Экспорт или импорт истории вычислений в формате CSV через кнопки в заголовке
- **Управление размером шрифта**: Настройка размера шрифта в 3 уровнях кнопками в нижнем левом углу.
- **Добавить заметки**: Добавляйте заметки к отдельным записям
- **Свайп влево (мобильный)**: Добавить/редактировать заметки
- **Свайп вправо (мобильный)**: Удалить записи

#### Управление настройками

- **Как открыть**: Клавиша F3 или боковое меню
- **Сброс**: Сброс всех настроек до значений по умолчанию
- **Экспорт/Импорт**: Сохранение или загрузка текущих настроек в формате JSON для использования одинаковых параметров в разных средах

#### Настройки отображения чисел

- **Применить формат чисел для каждого калькулятора**: Переключение клавишей Alt+n
- **Показать/Скрыть разделитель**: Переключение клавишей ,
- **Установить единицу группировки**: Переключение между 3/4 цифрами клавишей Alt+,
- **Десятичные знаки**: Настройка клавишами [, ] (без ограничений~16 цифр)

#### Использование горячих клавиш

- **Режим Shift**: Активация клавишей ' для доступа к расширенным функциям
- **Навигация по вкладкам**: Переход между вкладками Ctrl+Tab (→), Ctrl+Shift+Tab (←)
- **Переключение экранов**: F1 (Справка), F2 (О приложении), F3 (Настройки), F4 (История), F5 (Советы)

#### Копирование и вставка

- **Копировать основную панель**: Ctrl+C, Ctrl+Insert
- **Копировать дополнительную панель**: Shift+Ctrl+C, Alt+Ctrl+Insert
- **Вставить в основную панель**: Ctrl+V, Shift+Insert
- **Вставить в дополнительную панель**: Shift+Ctrl+V, Alt+Shift+Insert
- **Открыть меню**: Нажмите на панель и используйте меню для копирования/вставки

### **Советы по экспертным инструментам**

#### Математические функции

- **N-я степень/N-й корень**: Вычисление клавишами r/t
- **Тригонометрические функции**: Клавиши q/w/e в режиме shift
- **Извлечь целую/дробную часть**: Вычисление клавишами v/b
- **Факториал**: Вычисление клавишей h

#### Использование памяти

- **Сохранить и вызвать память**: Сохранение (MS) и вызов (MR) после вычисления
- **Вычисления с памятью**: Накопление значений функциями M+, M-, M×, M÷
- **Очистить память**: Очистка памяти с помощью MC
- **Статус памяти**: Нажмите на значок памяти на основной панели

#### Использование битовых операций

- **Битовые операции**: AND (j), OR (k), XOR (l), NOT (h), операции сдвига (r/t/u/i)
- **Логическое расширение**: Используйте операции NAND/NOR/XNOR клавишами q/w/e
- **Операции сдвига**: Перемещение битовых позиций на 1 бит/4 бита
- **Установить размер битов**: Битовые операции в соответствии с установленным размером

### **Оптимизация пользовательского опыта**

#### Макет экрана

- **Система тем**: Выбор из различных цветовых тем помимо тёмного/светлого режима (изменение в Настройках F3)
- **Поверх всех окон**: Переключение режима «поверх всех окон» клавишей Alt+t
- **Изменение размера окна**: Боковая панель автоматически подстраивается под размер окна
- **Сброс панели**: Переключение сброса панели при запуске клавишей Alt+i
- **Включить/выключить тёмный режим**: Переключение тёмного режима клавишей Alt+d
- **Применить формат чисел для каждого калькулятора**: Переключение формата чисел для каждого калькулятора клавишей Alt+n

#### Мобильная поддержка

- **Свайп**: Переключение режимов калькулятора свайпом влево или вправо
- **Включить/выключить тактильный режим**: Переключение тактильного режима клавишей Alt+p
- **Ландшафтный режим планшета**: Боковая панель автоматически подстраивается в ландшафтном режиме планшета

## Горячие клавиши (S: Shift, C: Control, A: Alt)

### Базовый калькулятор и общие функции

| Клавиша     | Функция                       |
| ----------- | ----------------------------- |
| 0-9.        | Ввод чисел и десятичной точки |
| +, -, \*, / | Базовая арифметика            |
| Enter, =    | Вычислить результат           |
| Backspace   | Удалить один символ           |
| Delete      | Сбросить калькулятор          |
| u           | Квадрат (x²)                  |
| i           | Квадратный корень (√x)        |
| j           | Сменить знак (±)              |
| k, %        | Процент (%)                   |
| l           | Обратное число (1/x)          |
| '           | Активировать режим shift      |

### Расширенные математические функции (режим Shift)

| Клавиша | Функция                     |
| ------- | --------------------------- |
| r       | Степень (xⁿ)                |
| t       | Корень (ⁿ√x)                |
| f       | Степень 10 (10ⁿ)            |
| g       | Остаток от деления (x%y)    |
| h       | Факториал (x!)              |
| q,w,e   | Триг. (sin, cos, tan)       |
| a,s,d   | Константы (Pi/2, ln10, ln2) |
| z,x,c   | Константы (Pi, phi, e)      |
| v       | Целая часть                 |
| b       | Дробная часть               |

### Операции с памятью

| Клавиша         | Функция                 |
| --------------- | ----------------------- |
| C-Delete        | Очистить память (MC)    |
| C-Backspace     | Вызвать память (MR)     |
| C-Enter, C-=    | Сохранить в память (MS) |
| C-+, C-Numpad + | Прибавить к памяти (M+) |
| C--, C-Numpad - | Вычесть из памяти (M-)  |
| C-_, C-Numpad _ | Умножить память (M×)    |
| C-/, C-Numpad / | Разделить память (M÷)   |

### Режим конвертации единиц/валют (режим Shift)

| Клавиша | Функция                                |
| ------- | -------------------------------------- |
| f,g,h   | ×2/×3/×5 или +5/+10/+100               |
| q,w,e   | ÷2/÷3/÷5 или -5/-10/-100               |
| a,s,d   | ×10/×100/×1000                         |
| z,x,c   | ÷10/÷100/÷1000                         |
| \       | Поменять источник и цель               |
| A-\     | Переключить отображение единицы/валюты |

### Режим конвертера систем счисления

| Клавиша | Функция                                   |
| ------- | ----------------------------------------- |
| r,t     | Сдвиг на 1 бит (x<<1, x>>1)               |
| u,i     | Сдвиг влево/вправо (x<<y, x>>y)           |
| f,g     | Сдвиг на 4 бита (x<<4, x>>4)              |
| h       | Операция NOT                              |
| j,k,l   | Битовые операции (AND, OR, XOR)           |
| q,w,e   | NAND, NOR, XNOR                           |
| z,x,c   | Ввод шестнадцатеричных (A, B, C)          |
| a,s,d   | Ввод шестнадцатеричных (D, E, F)          |
| \       | Поменять источник и цель                  |
| A-\     | Переключить отображение системы           |
| AC-\    | Переключить позицию символа (перед/после) |

| ↑/↓     | Просмотр истории выражений                               |
