#### Settings Management

- **How to Access**: F3 key or side menu
- **Reset**: Reset all settings to their default values
- **Export**: Save current settings to a JSON file
- **Import**: Load settings from a JSON file

#### Theme Settings

- **Preset Themes**: Select and apply from a variety of built-in themes
- **User Themes**: Create your own theme with the 'Create New Theme' button
- **Edit/Delete Themes**: User-created themes can be edited or deleted
