#### Parametres d'affichage des nombres

- **Appliquer le format de nombre par calculatrice** : Basculer avec la touche Alt+n. Utiliser des parametres de format de nombre independants pour chaque calculatrice (standard, unite, devise, base)
- **Afficher/Masquer le separateur** : Basculer avec la touche ,
- **Definir l'unite de regroupement** : Changer entre 3/4 chiffres avec la touche Alt+,
- **Decimales** : Ajuster avec les touches [, ] (illimite~16 chiffres)
