#### Convertisseur de devises

- **Comment y acceder** : Ctrl+3 ou selectionner l'onglet superieur
- **340 devises** : Fiduciaires, cryptomonnaies et métaux précieux — aucune clé API requise
- **Taux de change actuels** : Taux en temps réel (snapshot intégré utilisé hors ligne)
- **Favoris** : Definissez les devises frequemment utilisees comme favoris pour un acces rapide
- **Calcul rapide** : +5/+10/+100 (f/g/h), -5/-10/-100 (q/w/e)
- **Inverser les devises** : Utilisez la touche '\' pour inverser les devises source/cible
- **Symbole de devise** : Basculer l'affichage/masquage du symbole de devise avec la touche Alt+\
