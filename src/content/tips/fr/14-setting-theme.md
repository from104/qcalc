#### Gestion des parametres

- **Comment y acceder** : Touche F3 ou menu lateral
- **Reinitialiser** : Reinitialiser tous les parametres a leurs valeurs par defaut
- **Exporter** : Sauvegarder les parametres actuels dans un fichier JSON
- **Importer** : Charger les parametres depuis un fichier JSON

#### Parametres de theme

- **Themes predefinis** : Selectionner et appliquer parmi une variete de themes integres
- **Themes utilisateur** : Creer votre propre theme avec le bouton 'Creer un nouveau theme'
- **Modifier/Supprimer des themes** : Les themes crees par l'utilisateur peuvent etre modifies ou supprimes
