#### Настройки Отображения Чисел

- **Применить формат чисел для каждого калькулятора**: Переключите клавишей Alt+n. Используйте независимые настройки формата чисел для каждого калькулятора (базовый, единицы, валюты, системы счисления)
- **Показать/Скрыть разделитель**: Переключите клавишей ,
- **Установить единицу группировки**: Переключение между 3/4 цифрами клавишей Alt+,
- **Десятичные знаки**: Регулировка клавишами [, ] (без ограничений~16 цифр)
