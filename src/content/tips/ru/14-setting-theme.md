#### Управление Настройками

- **Как открыть**: Клавиша F3 или боковое меню
- **Сброс**: Сброс всех настроек до значений по умолчанию
- **Экспорт**: Сохранение текущих настроек в файл JSON
- **Импорт**: Загрузка настроек из файла JSON

#### Настройки Темы

- **Предустановленные темы**: Выберите и примените из множества встроенных тем
- **Пользовательские темы**: Создайте собственную тему с помощью кнопки «Создать новую тему»
- **Редактировать/Удалить темы**: Пользовательские темы можно редактировать или удалять
