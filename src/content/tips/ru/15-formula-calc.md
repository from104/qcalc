#### Калькулятор Формул

- **Как открыть**: Ctrl+5 или выберите из меню переполнения (⋯) в верхних вкладках
- **Ввод выражений**: Вводите математические выражения напрямую, используя синтаксис mathjs
- **Поддерживаемые операции**: Все математические функции mathjs, включая арифметику, степени, тригонометрию, логарифмы
- **Ссылка на текущее значение**: Используйте символ @ для включения текущего вычисленного значения в выражения
- **Функции памяти**: Ctrl+Enter (MS), Ctrl+Backspace (MR), Ctrl+Delete (MC)
- **Режим редактирования**: Нажмите Пробел или Enter (когда пусто) для входа, Esc для очистки и выхода
- **Навигация по истории**: Используйте клавиши со стрелками ↑/↓ в режиме редактирования для просмотра предыдущих выражений, автосброс при редактировании
- **Загрузить выражение**: Щёлкните правой кнопкой мыши по записи формулы → «Загрузить в поле формулы» для повторного использования выражений
