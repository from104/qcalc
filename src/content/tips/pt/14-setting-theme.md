#### Gerenciamento de Configurações

- **Como Acessar**: Tecla F3 ou menu lateral
- **Redefinir**: Redefinir todas as configurações para os valores padrão
- **Exportar**: Salvar as configurações atuais em um arquivo JSON
- **Importar**: Carregar configurações de um arquivo JSON

#### Configurações de Tema

- **Temas Predefinidos**: Selecione e aplique entre uma variedade de temas integrados
- **Temas do Usuário**: Crie seu próprio tema com o botão 'Criar Novo Tema'
- **Editar/Excluir Temas**: Temas criados pelo usuário podem ser editados ou excluídos
