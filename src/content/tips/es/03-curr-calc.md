#### Convertidor de Divisas

- **Cómo acceder**: Ctrl+3 o seleccionar la pestaña superior
- **340 divisas**: Fiat, criptomonedas y metales preciosos — sin clave API necesaria
- **Tipos de cambio actualizados**: Tipos en tiempo real (se usa una instantánea integrada sin conexión)
- **Favoritos**: Establece las divisas de uso frecuente como favoritas para un acceso rápido
- **Cálculo rápido**: +5/+10/+100 (f/g/h), -5/-10/-100 (q/w/e)
- **Intercambiar divisas**: Usa la tecla '\' para intercambiar las divisas de origen/destino
- **Símbolo de divisa**: Alterna mostrar/ocultar el símbolo de divisa con la tecla Alt+\
