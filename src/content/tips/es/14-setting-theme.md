#### Gestión de Configuración

- **Cómo acceder**: Tecla F3 o menú lateral
- **Restablecer**: Restablece todas las configuraciones a sus valores predeterminados
- **Exportar**: Guarda la configuración actual en un archivo JSON
- **Importar**: Carga la configuración desde un archivo JSON

#### Configuración de Temas

- **Temas predefinidos**: Selecciona y aplica entre una variedad de temas integrados
- **Temas de usuario**: Crea tu propio tema con el botón 'Crear nuevo tema'
- **Editar/Eliminar temas**: Los temas creados por el usuario se pueden editar o eliminar
