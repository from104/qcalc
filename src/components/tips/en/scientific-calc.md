##### Scientific Calculator Features

- Trigonometric functions:
  - sin/cos/tan: **q/w/e** keys
  - Inverse functions in Shift mode
- Mathematical constants:
  - π(pi): **z** key
  - e(euler): **c** key
  - φ(phi): **x** key
  - π/2, ln10, ln2: **a/s/d** keys
- Advanced operations:
  - xⁿ(power): **r** key
  - ⁿ√x(root): **t** key
  - 10ⁿ: **f** key
  - x!: **h** key
- Integer/decimal extraction: **v/b** keys
