# qcalc 작업 목록

## 🥇 우선순위: 높음

- [ ] **결과 음성 출력 (TTS)**
  - **설명**: 계산 결과를 소리로 읽어주는 기능 (접근성).
  - **구현**: Web Speech API (`speechSynthesis`) 연동, 상태 관리.
- [ ] **사용자 테마 설정**
  - **설명**: 사용자가 직접 색상을 선택하여 자신만의 테마를 만들고 저장.
  - **구현**: 설정 페이지에 컬러 피커 UI 추가, LocalStorage에 저장.
  - **세부 명세**:
    - [ ] Quasar 색상 팔레트 이름('red-1', 'grey-5' 등) 사용.
    - [ ] Firebase 연동으로 테마 공유 커뮤니티 기능 구현.

---

## 🥈 우선순위: 중간

- [ ] **모바일 앱 구현 (Capacitor: iOS)**
  - **설명**: iOS 플랫폼용 앱 빌드 및 배포.
  - **구현**: Apple 개발자 계정 설정, Xcode 연동, iOS 네이티브 기능 처리.
- [ ] **계산기 음성 조작 (STT)**
  - **설명**: 음성 명령으로 계산기 조작 (접근성).
  - **구현**: Web Speech API (`SpeechRecognition`) 연동, 자연어 처리.

---

## 🥉 우선순위: 낮음

- [ ] **스킨 기능**
  - **설명**: UI 레이아웃을 통째로 변경하는 기능.
  - **구현**: 스킨 관리 아키텍처 설계.
- [ ] **광고 기능 (수익화)**
  - **설명**: 배너 광고 추가.
  - **구현**: AdMob 등 광고 SDK 연동.
