# qcalc 개발 일지

이 파일은 qcalc 프로젝트의 개발 과정을 기록하는 작업 일지입니다. 날짜별로 수행한 주요 작업, 버그 수정, 리팩토링 내역을 기록하여 프로젝트의 진행 상황을 추적하고 관리합니다.

## 작성법

- 날짜별로 섹션을 생성합니다. (예: `## 2025-10-08`)
- 각 작업 항목은 글머리 기호(-)를 사용하여 간결하게 작성합니다.
- 완료된 작업은 `[x]` 체크박스를 사용하여 표시할 수 있습니다.
- 관련 커밋이 있다면 링크를 추가할 수 있습니다.
